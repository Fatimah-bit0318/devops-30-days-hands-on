# GitHub Workflow

```text
Create Repository
        ↓
Clone Repository
        ↓
Create New Branch
        ↓
Modify Code
        ↓
Commit Changes
        ↓
Push Branch
        ↓
Create Pull Request
        ↓
Review
        ↓
Merge
```

### Useful Commands

```bash
git remote add origin <repo-url>

git push -u origin main

git fetch

git pull

git checkout -b feature/linux-notes
```
